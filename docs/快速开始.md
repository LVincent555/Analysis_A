# 股票分析系统 - 快速开始指南

## 🚀 一键启动

### 方式一：使用启动脚本（推荐）

1. **启动后端**
   - 双击运行 `start_backend.bat`
   - 等待提示 "Uvicorn running on http://0.0.0.0:8000"
   - 后端API启动成功！

2. **启动前端**
   - 双击运行 `start_frontend.bat`  
   - 等待浏览器自动打开 http://localhost:3000
   - 前端应用启动成功！

### 方式二：手动启动

#### 后端启动
```bash
cd backend
pip install -r requirements.txt  # 首次运行需要安装依赖
python main.py
```

#### 前端启动
```bash
cd frontend
npm install  # 首次运行需要安装依赖
npm start
```

## 📖 使用说明

### 1. 访问应用
启动后，在浏览器打开：http://localhost:3000

### 2. 功能说明

**板块类型选择**
- **主板（排除双创）**: 只分析主板股票，排除 300/301/688/920 开头的股票
- **全部（含双创）**: 包含所有股票，含创业板和科创板

**分析周期选择**
- **2天**: 股票必须在最近2天都出现
- **3天**: 股票必须在最近3天都出现  
- **5天**: 股票在最新一天出现，且在最近5天内至少再出现1次
- **7天**: 股票在最新一天出现，且在最近7天内至少再出现1次
- **14天**: 股票在最新一天出现，且在最近14天内至少再出现1次

### 3. 查看结果

分析结果表格显示：
- **排名**: 按最新日期的排名从小到大排序
- **股票代码**: 6位股票代码
- **出现次数**: 在分析周期内出现的总次数
- **最新排名**: 在最新日期表格中的排名
- **出现日期及排名**: 每次出现的具体日期和排名

## 📊 数据要求

数据文件需放在项目上一级目录，文件名格式：
```
YYYYMMDD_data_sma_feature_color.xlsx
```

例如：
- 20251103_data_sma_feature_color.xlsx
- 20251104_data_sma_feature_color.xlsx  
- 20251105_data_sma_feature_color.xlsx

## ⚠️ 注意事项

1. **首次运行**: 需要等待依赖安装完成（可能需要几分钟）
2. **Excel文件**: 分析前请关闭所有Excel数据文件
3. **端口占用**: 如果8000或3000端口被占用，请先关闭占用的程序
4. **数据更新**: 添加新数据文件后，点击"刷新数据"按钮重新分析

## 🆘 常见问题

### Q: 后端启动失败？
A: 检查Python版本（需要3.8+），确保pip可以正常使用

### Q: 前端启动失败？
A: 检查Node.js版本（需要16+），确保npm可以正常使用

### Q: 显示"未找到数据文件"？
A: 确保数据文件在正确的位置（项目上一级目录）

### Q: 前端无法连接后端？
A: 确保后端已启动，并且运行在 http://localhost:8000
