## 版本 0.1.1 功能亮点

### 🎯 核心功能：正态分布±σ筛选

从海量股票中智能筛选核心标的，提供5档可调σ倍数，实现逐级精准筛选。

#### 📊 5档σ倍数选项

| 档位 | σ倍数 | 覆盖范围 | 筛选效果 | 适用场景 |
|------|-------|----------|----------|----------|
| **±1.0σ** | 标准 | 约68% | 宽松筛选 | 全面了解分布 |
| **±0.75σ** | 较窄 | 约55% | 过滤边缘 | 缩小候选范围 |
| **±0.5σ** | 窄 | 约38% | 聚焦中等 | 常用筛选档位 |
| **±0.3σ** | 很窄 | 约24% | 精选核心 | 深度筛选 |
| **±0.15σ** | 极窄 | 约12% | 极度精选 | 最严格筛选 |

#### 📈 实测数据

**排名跳变（阈值2500）**
- 初始总数：320只
- ±1.0σ → 204只（63.8%）
- ±0.5σ → 101只（31.6%）
- ±0.15σ → 28只（8.8%）

**稳步上升（3天）**
- 初始总数：508只
- ±1.0σ → 325只（64.0%）
- ±0.5σ → 173只（34.1%）
- ±0.15σ → 60只（11.8%）

## 快速开始

1. **启动后端**
   ```bash
   cd backend
   python -m uvicorn app.main:app --reload --port 8000
   ```

2. **启动前端**
   ```bash
   cd frontend
   npm start
   ```

3. **访问应用**
   - 前端：http://localhost:3000
   - 后端API文档：http://localhost:8000/docs
