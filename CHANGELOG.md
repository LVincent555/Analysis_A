# 更新日志

## [0.1.1] - 2025-11-06

### 新增功能
- **正态分布±σ筛选功能**
  - 排名跳变模块新增±σ范围筛选
  - 稳步上升模块新增±σ范围筛选
  - 提供5档可调σ倍数选项：±1.0σ、±0.75σ、±0.5σ、±0.3σ、±0.15σ
  - 从宽到窄逐级筛选，支持从500+只股票缩减到20-30只
  
### 功能改进
- 添加统计信息展示：均值、标准差、±σ范围
- 实时显示σ范围内股票数量
- σ倍数切换按钮组，直观易用
- 支持与排序功能无缝配合

### 后端改进
- API新增`sigma_multiplier`参数
- 自动计算±σ范围并筛选股票
- 优化缓存策略，包含σ倍数参数

### 前端改进
- 新增σ倍数选择按钮组（5档）
- 动态显示筛选结果数量
- 优化UI布局和交互体验

### 测试验证
- 排名跳变（阈值2500）：320只 → 204只(±1σ) → 28只(±0.15σ)
- 稳步上升（3天）：508只 → 325只(±1σ) → 60只(±0.15σ)
